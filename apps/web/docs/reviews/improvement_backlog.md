# 개선 사항 백로그 (Improvement Backlog)

아래는 정밀 감사 결과 도출된 실질적 개선 작업 항목들입니다.

## 백로그 항목 상세

### [P0] 유출된 운영 연동 Credentials 전면 무효화 및 재발급
* **제목**: 노출된 외부 연동 API 시크릿 갱신 및 로테이션
* **설명**: `application-prod.yml`에 하드코딩되었던 AWS, Google, Discord, VAPID 및 JWT 토큰의 실제 유출된 시크릿 키들을 즉시 무효화 조치하고 새로운 비밀값으로 교체하십시오.
* **기대 효과**: 외부 사칭 및 리소스 오남용 차단.
* **우선순위**: P0
* **작업 난이도**: 하
* **예상 영향력**: 최상 (인프라 보안 복구)
* **선행 조건**: 없음

---

### [P1] 환경변수를 활용한 설정 정보 관리 자동화
* **제목**: 운영 설정 파일 내 시크릿 환경변수 치환
* **설명**: `application-prod.yml`에 있는 중요 보안 상수를 `${JWT_SECRET}` 등의 스프링 환경변수 기입식으로 수정하고, 이를 Docker Compose 파일에서 주입받도록 구성합니다.
* **기대 효과**: 보안 위험 원천 소멸.
* **우선순위**: P1
* **작업 난이도**: 하
* **예상 영향력**: 상
* **선행 조건**: [P0] 시크릿 교체 완료

---

### [P1] JPA 소프트 딜리트 글로벌 필터 자동화 적용
* **제목**: 엔티티 클래스 단위 Soft-Delete Restriction 선언
* **설명**: `User`, `CourseReview` 등 삭제 이력이 발생하는 엔티티 상단에 `@SQLRestriction` 지시문을 명시적으로 바인딩하여, 비즈니스 쿼리 작성 시 삭제 사용자가 노출되는 현상을 자동화 차단합니다.
* **기대 효과**: 불필요한 논리 삭제 데이터 노출 사고 방지.
* **우선순위**: P1
* **작업 난이도**: 중
* **예상 영향력**: 상
* **선행 조건**: 없음

---

### [P2] DB 복합 조회 인덱스 스크립트 작성
* **제목**: 과목 검색 및 리뷰 동적 조건 튜닝용 인덱싱 스크립트 배포
* **설명**: Flyway 마이그레이션 스크립트를 통해 `courses(subject_code, professor)` 복합 인덱스를 적용합니다.
* **기대 효과**: 강의 평점 연계 검색 속도 향상.
* **우선순위**: P2
* **작업 난이도**: 하
* **예상 영향력**: 중
* **선행 조건**: 없음

---

### [P2] API 공통 응답 규격 래핑 동기화
* **제목**: 미적용 컨트롤러 응답 시 CommonResponse 래퍼 일괄 적용
* **설명**: `AdminFeedbackController` 등 날것의 구조를 반환하는 메서드들을 `ResponseEntity<CommonResponse<T>>`로 전면 리팩토링합니다.
* **기대 효과**: 프론트-백 간 통합 연계 통신 코드 단순화.
* **우선순위**: P2
* **작업 난이도**: 중
* **예상 영향력**: 중
* **선행 조건**: 없음

---

### [P2] 대용량 페이지네이션 수신 한계값 강제 설정
* **제목**: API Pageable Max Size 제한 어댑터 적용
* **설명**: 수강평 및 피드백 페이지네이션 조회 엔드포인트에 `Pageable` 크기가 50을 초과하지 않도록 보완 로직을 적용합니다.
* **기대 효과**: 대량 데이터 조회 시의 영속성 컨텍스트 과부하 및 메모리 크래시 방지.
* **우선순위**: P2
* **작업 난이도**: 하
* **예상 영향력**: 중
* **선행 조건**: 없음
