# 주요 기능 업데이트 (Feature Updates - Web)

이 문서는 `web` 모듈의 버전별 주요 기능 추가 및 변경 사항을 기록합니다.

---

## 전북대 수강신청 도우미 플랫폼 공식 릴리스 (v1.0.0)

풀스택 및 반응형 웹 기술과 고도화된 시스템 시각화가 집약된 정식 버전입니다.

### 1. 지연 없는 강의 조회 및 무한 스크롤 검색 시스템 ⚡

- **Infinite Query 리팩토링**: `@tanstack/react-query`의 `useInfiniteQuery` 및 `Intersection Observer API`를 조립해, 수천 건의 강의 리스트에 대해 끊김과 멈춤 없는 자동 페이지 로드 사용자 경험을 제공.
- **백엔드 연동 통합 찜 필터**: 단순히 UI 단에서 감추는 수준이 아닌 쿼리스트링 기반으로 실제 서버 DB 조회를 수행하는 **'찜한 강의만'** 토글 기능 구축. 비로그인 유저 조작 시 브라우저 인터셉트 및 로그인 안내(`Toast`).
- **메타 데이터 가독성 정규화**: 학과명(`컴퓨터공학부 3` -> `컴퓨터공학부`), 수강 대상 등의 정보에 존재하는 시각적 노이즈 스페이스나 오기입 학년 숫자, 전체 표기 등을 정규표현식으로 정리해 깔끔한 레이아웃 보장.

### 2. 고성능 사용자 인증 및 학생 온보딩 워크플로우 🎓

- **단계별 온보딩 시퀀스**: 신규 사용자가 가장 빈도 높게 설정해야 하는 알림 수신 동의 여부(푸시, 디스코드 등)와 필수 연락망 입력을 쉽게 마칠 수 있게 다이얼로그 기반의 단계형 온보딩 화면을 제공해 이탈률을 억제.
- **퍼스널 푸시 채널 설정 (마이페이지)**: 웹 PWA(FCM 기반 모바일 푸시), 이메일, 그리고 즉각적인 Discord 통합 알림 스위치를 유저가 자유롭게 토글(켜고 끄기)할 수 있는 직관적 관리 UI.
- **비인가 및 권한 가드**: Admin 전용, 세션 만료 등의 시나리오에서 사용자 진입을 안전하게 차단하고 `/login`으로 리다이렉션.

### 3. 세련된 반응형 컴포넌트 환경 🎨

- **단일 다이얼로그 모듈화**: 강의 세부 사항 명세 창을 별도 라우트 페이지로 분리하지 않고 공통 컴포넌트(`CourseDetailContent`) 다이얼로그 형태로 호출하여 스크롤 오버헤드나 흐름 단절을 철저히 차단.
- **다크 모드 & 시각적 일관성**: Tailwind CSS를 사용한 우수한 다크 테마 대응. 강좌 유효 상태(모집 중, 마감 등)에 따른 컬러 Badge(`bg-red-50 text-red-600` 등).
- **빈자리 구독 보호**: 크롤링 중인 최신 상태와 일치하지 않는 옛날 강의에서 불필요한 알림 구독 행위를 방지하는 UX (비활성화 및 툴팁 표기, "조회 전용" 뱃지 추가 탑재).

### 4. 실전 지향적 수강신청 유틸리티 지원 🏫

- **모의 시뮬레이터(`TimetableGrid`)**: 실제 과목의 수강 시간을 타임테이블 블록으로 겹쳐볼 수 있는 위젯. 블록 렌더링 로직의 재사용성을 위해 전용(`TimetableBlock`) 컴포넌트로 책임 분리.
- **인라인 캠퍼스 카카오 지도 매핑**: 강의실 문자열(예: "공대 4호관 401호")이 모호한 재학생 및 신입생들의 길찾기를 돕기 위한 카카오맵 비동기 SDK 임베딩.
- **스마트 키워드 치환 엔진**: "자대" -> "자연과학대학", "인대" -> "인문대학" 등의 약어 키워드를 실제 지도가 이해할 수 있게 자동 정규화.

### 5. 고도화된 메인 웹 포털 포지션 📈

- **실시간 유저 대시보드 (`HomeLanding.tsx`)**: 히어로 섹션 아래에서 실시간으로 추천 및 조회수가 높은 강의 순위를 보여주는 트렌드 스와이퍼/컨테이너 배치.
- **학사 일정 D-Day 뱃지 렌더**: `useUpcomingSchedules` Hook 연결로 지연 로딩 표시(`Loader2`) 처리. 오늘 기준 지난 일정은 숨기고 다가올 이벤트들을 명확히 띄워주는 다이내믹 캘린더 피드.

### 6. 전지적 어드민 통합 시각화 워크스페이스 🛠️

- **크롤링 타겟 인하우스 제어 (`AdminCrawlTargetPanel`)**: 터미널이나 서버 배포 없이, 대시보드에서 GUI로 목표 연도와 1, 2학기/계절학기 정보 등을 골라 즉각적으로 실시간 스크래핑을 실행하고 저장하는 폼.
- **실시간 시스템/트래픽 통계 대시보드**:
  - `AdminTrafficChart`: 최근 가입자, 이용자, 알림 발송건 증감을 Recharts(SVG 라인/에어리어 차트) 라이브러리를 활용해 유연하게 시각화.
  - `AdminStatCard`: 전체 강의 개수 조망 등 시스템 개요 번역 및 즉시 모니터링(`AdminOverview`).
- **서버 활동 로그 추적기 (`AdminActivityLog`)**: 서버 구동/에러/크롤링 결과 등의 실시간 이벤트를 시간 역순 타임라인으로 피드백하여 볼 수 있도록 통합.
- **일정 캘린더 In-place 조작판 (`AdminSchedulePanel`)**: 메인 포탈에 보일 주요 행사를 CRUD(`POST, DELETE, PUT`) 형태로 자유롭게 편집 및 삭제, `QueryClient.invalidateQueries`를 통한 즉각적 무결성 반영.
- **엄격한 프론트엔드 타입 체커**: React Query 콜백에 `ScheduleResponse` 등 명시적 서버 DTO 구조체를 완전 귀속시켜 컴파일 타임 에러 `any`를 박멸하는 컨벤션 규칙 확보.
