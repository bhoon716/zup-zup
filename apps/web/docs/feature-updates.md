# 주요 기능 업데이트 (Feature Updates - Web)

이 문서는 `web` 모듈의 버전별 주요 기능 추가 및 변경 사항을 기록합니다.

---

## 주요 기능 업데이트 (v1.1)

### 강의 검색 무한 스크롤 (Infinite Scroll)

- **Infinite Query**: `@tanstack/react-query`의 `useInfiniteQuery`를 사용하여 페이지 단위 데이터 패칭을 구현했습니다.
- **Intersection Observer**: 스크롤 하단을 감지하여 자동으로 다음 페이지를 불러오는 끊김 없는 사용자 경험을 제공합니다.
- **UI 간소화**: 검색 결과 상단의 카운터 표시를 제거하고, 로딩 인디케이터를 개선하여 몰입도를 높였습니다.

---

## 주요 기능 업데이트 (v1.3)

### 백엔드 통합 찜 필터링 (Server-side Wishlist Filter)

- **검색 엔진 통합**: 클라이언트 사이드 필터링의 한계를 극복하기 위해 Querydsl 서브쿼리를 이용한 백엔드 통합 검색 시스템을 구축했습니다.
- **스마트 UI**: 검색 바에 **'찜한 강의만'** 토글 버튼을 추가하고, 비로그인 유저를 위한 접근 제어(로그인 유도 Toast) 로직을 적용했습니다.
- **성능 최적화**: 수천 개의 강의 데이터 중 내가 찜한 강의만 즉시 필터링하여 검색 속도와 데이터 정합성을 동시에 확보했습니다.

---

## 주요 기능 업데이트 (v1.4)

### 컴포넌트 리팩토링 및 코드 품질 개선 (Component Refactoring)

- **UI 재사용성 강화**: 중구난방으로 흩어져 있던 강의 상세 정보 UI를 `CourseDetailContent` 공통 컴포넌트로 통합하여 `CourseDetailDialog`와 `TimetableGrid`에서 동일한 프리미엄 UI를 제공하도록 개선했습니다.
- **클린 코드 및 관심사 분리**: `TimetableGrid` 내부에 통합되어 있던 복잡한 블록 렌더링 로직을 `TimetableBlock` 컴포넌트로 분리하여 가독성을 높이고 유지보수성을 확보했습니다.
- **다크 모드 및 시각적 일관성**: 모든 상세 정보 영역에 일관된 다크 모드 스타일과 레이아웃을 적용하여 시각적 완성도를 높였습니다.

---

## 주요 기능 업데이트 (v1.5)

### 카카오맵 연동 및 실시간 강의실 위치 정보 (Campus Map Integration)

- **인라인 지도 임베드**: 강의 상세 다이얼로그에서 강의실 위치를 카카오맵 지도로 즉시 확인할 수 있는 기능을 추가했습니다.
- **지능형 키워드 매핑**: 복잡한 강의실 텍스트(예: "공대 4호관 401호")를 자동 정규화하여 지도 검색 정확도를 극대화했습니다.
- **외부 지도 연동**: 상세 확인이 필요한 경우를 위해 네이버 지도 및 카카오 지도 앱/웹으로 바로 연결되는 외부 링크를 제공합니다.
- **안전한 SDK 로딩**: Next.js 환경에 최적화된 비동기 SDK 로더를 통해 페이지 성능 저하 없이 안정적으로 지도를 렌더링합니다.

---

## 주요 기능 업데이트 (v1.7)

### 강의 정보 디스플레이 최적화 (Course Info Display Optimization)

- **학과명 데이터 정규화**: 학과명 뒤에 붙는 중복된 학년 숫자나 불필요한 공백을 제거하여 시각적 노이즈를 축소했습니다. (예: "컴퓨터공학부 3" → "컴퓨터공학부")
- **학년 표시 정제**: 학과명과 대상 학년이 나란히 표시될 때 "전체" 학년인 경우 표시를 생략하여 정보 밀도를 최적화했습니다.
- **가독성 향상**: 상세 페이지 상단의 메타 정보 레이아웃을 더 깔끔하게 정리했습니다.
